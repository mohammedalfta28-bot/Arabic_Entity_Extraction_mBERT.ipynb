# 🔍 نظام استخراج الكيانات العربية (Arabic Named Entity Recognition)

هذا المشروع هو تطبيق متقدم في معالجة اللغات الطبيعية (NLP) يهدف إلى استخراج وتصنيف المعلومات الهامة من النصوص العربية الخام وتحويلها إلى بيانات مهيكلة.

## 📝 وصف المشروع
يقوم النظام بتحليل الجمل العربية والتعرف تلقائياً على ثلاثة أنواع رئيسية من الكيانات:
1. **الأشخاص (PER):** الأسماء الشخصية.
2. **المواقع (LOC):** المدن، الدول، والمعالم الجغرافية.
3. **المنظمات (ORG):** الشركات، الجامعات، والمؤسسات.

## 🤖 النموذج المستخدم (Model)
يعتمد المشروع على نموذج **Multilingual BERT (mBERT)** المخصص لتصنيف الكلمات (Token Classification):
`Davlan/bert-base-multilingual-cased-ner-hrl`
تم اختيار هذا النموذج لقدرته العالية على معالجة خصوصية اللغة العربية ودقته في فهم السياق اللغوي.

## 🚀 كيفية التشغيل
1. ارفع ملف الـ `.ipynb` إلى بيئة **Google Colab**.
2. قم بتفعيل المعالج الرسومي **GPU** لضمان سرعة الاستجابة.
3. قم بتشغيل الخلايا بالترتيب؛ سيقوم الكود بتثبيت المكتبات اللازمة وتحميل النموذج تلقائياً.

## 📊 مثال لنتائج التشغيل
**النص:** "سافر محمد محمد الأسود من دبي إلى الرياض لزيارة شركة أرامكو"
| الكيان | النوع |
| :--- | :--- |
| محمد محمد الأسود | شخص (PER) |
| دبي | مكان (LOC) |
| الرياض | مكان (LOC) |
| أرامكو | منظمة (ORG) |

---
**إعداد الطالب:** محمد محمد الأسود  
**التخصص:** هندسة الحاسوب والذكاء الاصطناعي
